# 运行项目 在项目根目录执行

vuepress dev

# 打包项目 在项目根目录执行

vuepress build

# 发布项目到github 可直接观看

  1. 打包后会新创建 /.vuepress/dist目录
  2. 将 '/.vuepress/dist目录' 改名为 'docs' 放到项目根目录下
  3. git push 就可以了
  4. 访问地址 https://xueyueshuai.github.io/docs 可能要过点时间才能看到结果 一般为 3-4分钟
